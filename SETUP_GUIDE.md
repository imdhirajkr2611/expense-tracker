# 🚀 Expense Tracker - Complete Setup Guide for Windows/Mac/Linux\n\n## Prerequisites\n\nYou need to install 3 things:\n\n1. **XAMPP** (PHP + MySQL Server)\n2. **Git** (to clone the repository)\n3. **Code Editor** (VS Code recommended)\n\n---\n\n## Step 1: Download & Install XAMPP\n\n### For Windows:\n1. Go to https://www.apachefriends.org/download.html\n2. Download **XAMPP for Windows** (PHP 7.4 or higher)\n3. Run the installer (.exe file)\n4. Choose installation folder (default is fine)\n5. Select components: Make sure **Apache** and **MySQL** are checked\n6. Complete the installation\n7. XAMPP Control Panel will open automatically\n\n### For Mac:\n1. Download **XAMPP for macOS** from https://www.apachefriends.org/\n2. Open the .dmg file\n3. Drag XAMPP to Applications folder\n4. Open Applications > XAMPP folder\n5. Double-click xampp-control.app\n\n### For Linux:\n```bash\n# Ubuntu/Debian\nwget https://www.apachefriends.org/xampp-files/8.2.12/xampp-linux-8.2.12-installer.run\nchmod +x xampp-linux-8.2.12-installer.run\nsudo ./xampp-linux-8.2.12-installer.run\n```\n\n---\n\n## Step 2: Start XAMPP Services\n\n### Windows/Mac:\n1. Open **XAMPP Control Panel**\n2. Click **Start** button next to **Apache**\n3. Click **Start** button next to **MySQL**\n4. Both should show green indicators\n\n### Linux:\n```bash\nsudo /opt/lampp/lampp start\n```\n\n---\n\n## Step 3: Clone the Repository\n\n### Option A: Using Git (Recommended)\n\n1. **Install Git** from https://git-scm.com/download\n\n2. **Open Terminal/Command Prompt:**\n\n   **Windows:** Right-click on folder > \"Open PowerShell here\" or \"Open Command Prompt here\"\n   \n   **Mac/Linux:** Open Terminal\n\n3. **Navigate to htdocs:**\n   ```bash\n   # Windows\n   cd \"C:\\xampp\\htdocs\"\n   \n   # Mac\n   cd /Applications/XAMPP/xamppfiles/htdocs\n   \n   # Linux\n   cd /opt/lampp/htdocs\n   ```\n\n4. **Clone the repository:**\n   ```bash\n   git clone https://github.com/imdhirajkr2611/expense-tracker.git\n   cd expense-tracker\n   ```\n\n### Option B: Download ZIP (No Git)\n\n1. Go to https://github.com/imdhirajkr2611/expense-tracker\n2. Click **Code** button (green button)\n3. Click **Download ZIP**\n4. Extract the ZIP file to your htdocs folder:\n   ```\n   C:\\xampp\\htdocs\\expense-tracker  (Windows)\n   /Applications/XAMPP/xamppfiles/htdocs/expense-tracker  (Mac)\n   /opt/lampp/htdocs/expense-tracker  (Linux)\n   ```\n\n---\n\n## Step 4: Create the Database\n\n### Method 1: Using phpMyAdmin (Easiest)\n\n1. Open your browser and go to: **http://localhost/phpmyadmin**\n\n2. Click on **\"New\"** (left sidebar) or **\"+ New\"** button\n\n3. Enter database name: `expense_tracker`\n\n4. Click **Create**\n\n5. Click on the new `expense_tracker` database\n\n6. Click on **\"Import\"** tab\n\n7. Click **\"Choose File\"** and select `database/schema.sql` from your project folder\n\n8. Click **Import**\n\n✅ Database created with sample data!\n\n### Method 2: Using Terminal/Command Line\n\n```bash\n# Windows (PowerShell)\ncd C:\\xampp\\mysql\\bin\n.\\mysql -u root\n\n# Mac/Linux\n/Applications/XAMPP/xamppfiles/bin/mysql -u root\n# or\n/opt/lampp/bin/mysql -u root\n```\n\nThen paste these commands:\n```sql\nCREATE DATABASE expense_tracker;\nUSE expense_tracker;\n-- Then copy & paste the contents of database/schema.sql\n```\n\n---\n\n## Step 5: Configure Database Connection\n\n1. Open `api/config.php` in your code editor\n2. Update these lines:\n\n```php\ndefine('DB_HOST', 'localhost');    // Keep this\ndefine('DB_USER', 'root');          // Usually 'root' for XAMPP\ndefine('DB_PASS', '');              // Leave empty for XAMPP (no password by default)\ndefine('DB_NAME', 'expense_tracker'); // Should match database name\n```\n\n**Note:** If you set a MySQL password during XAMPP installation, replace the empty string with your password.\n\n---\n\n## Step 6: Run the Application\n\n### Make sure XAMPP is running (Apache + MySQL both green)\n\n1. **Open your browser**\n2. **Go to:** `http://localhost/expense-tracker/`\n3. **You should see the Expense Tracker UI!** 🎉\n\n---\n\n## Troubleshooting\n\n### \"Cannot connect to database\" error\n\n**Solution:**\n- Check if MySQL is running in XAMPP (green indicator)\n- Verify credentials in `api/config.php`\n- Check if `expense_tracker` database exists in phpMyAdmin\n\n### \"Page not found\" error\n\n**Solution:**\n- Make sure you extracted files to the correct htdocs folder\n- Check the URL is: `http://localhost/expense-tracker/`\n- Not: `http://localhost/index.html` or `http://localhost/expense-tracker/index.html`\n\n### \"API calls not working\" (blank charts)\n\n**Solution:**\n- Open browser **F12** (Developer Tools)\n- Go to **Console** tab\n- Check for error messages\n- The app has a demo mode - it still works with mock data\n\n### \"Port 80 or 3306 already in use\"\n\n**Solution:** Change XAMPP port:\n1. Open XAMPP Control Panel\n2. Click \"Config\" button next to Apache\n3. Edit port in apache config file\n4. Restart Apache\n\n---\n\n## Using the Application\n\n### Add an Expense:\n1. Select a **Category** (Food, Transport, etc.)\n2. Enter **Amount**\n3. Enter **Description**\n4. Select **Date**\n5. Click **Add Expense**\n\n### View Charts:\n- **Left chart (Pie/Doughnut):** Shows spending by category\n- **Right chart (Bar):** Shows monthly spending trends\n\n### Filter Expenses:\n1. Use the **Filter by Category** dropdown\n2. Click **Clear Filters** to reset\n\n### Delete an Expense:\n- Click the **Delete** button in the table\n- Confirm deletion\n\n---\n\n## Project File Structure\n\n```\nexpense-tracker/\n├── index.html                 # Main page (open this in browser)\n├── css/\n│   └── style.css              # Styling\n├── js/\n│   └── app.js                 # Frontend JavaScript + Chart.js\n├── api/\n│   ├── config.php             # Database configuration (EDIT THIS)\n│   ├── expenses.php           # API for expense CRUD\n│   └── charts.php             # API for chart data\n├── database/\n│   └── schema.sql             # Database schema (import this)\n├── README.md                  # Project info\n├── SETUP_GUIDE.md            # This file\n└── .gitignore                # Git ignore file\n```\n\n---\n\n## VS Code Setup (Optional)\n\nMake development easier:\n\n1. **Install VS Code** from https://code.visualstudio.com/\n2.
